# 简介

- 横向和纵向的刻度尺

# API  

| 名称 | 说明 | 类型 | 默认值 |
| --- | --- | --- | --- |
| padding | 比如传了 `15`，那就会被转换成 `padding: 15`；比如传了 `15px`，那就会被转换成 `padding: 15px` | `string | number` | - |

# 示例

``` jsx
    import Ruler from '.'

    render = () => <Ruler>test</Ruler>
```
